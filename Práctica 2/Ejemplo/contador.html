CA-SegI

Autoridad Certificadora para la materia Seguridad Informática

Esta autoridad certficadora ha sido desarrollada para unas prácticas de la materia Seguridad Informática de la USC. No tiene ninguna validez legal. Si no es un alumno de esta materia, por favor no emplee este servicio.

Para emplear los certificados emitidos por esta CA deberá en primer lugar instalar el certificado raíz de la misma en su navegador, gestor de correo o programa en el que vaya a utilizar dichos certificados.

Certificado raíz de esta CA en formato PEM: certificado

Instrucciones para instalarlo en Internet Explorer: Herramientas -> Opciones de internet -> Contenido -> Certificados -> Entidades emisoras raíz de confianza -> Importar...
Instrucciones para instalarlo en Firefox: Herramientas -> Opciones -> Avanzado -> Cifrado -> Ver certificados -> Autoridades -> Importar ... ó Editar -> Preferencias -> Avanzado -> Cifrado -> Ver certificados -> Autoridades -> Importar ...
Instrucciones para instalarlo en Thunderbird: Herramientas -> Opciones -> Privacidad -> Seguridad -> Administrar certificados... -> Autoridades -> Importar ...


El segundo paso será crear una solicitud de certificado y enviarla a la CA para que la firme y nos devuelva el certificado, según las instrucciones recibidas. En el plazo más breve posible la CA firmará dicho certificado y éste se publicará en esta página web (ver lista a continuación) para que tanto su propietario como cualquier otra persona lo pueda descargar.



Lista de certificados de usuario























A partir de vuestro propio certificado y de vuestra propia clave privada podéis generar un certificado en formato pkcs#12 (extensión .p12), que es el más común para muchas aplicaciones. Queda entonces el último paso: incluir vuestro certificado digital en el navegador o gestor de correo para poder cifrar y firmar con él.

Instrucciones para incluir un certificado personal en el Internet Explorer: Herramientas -> Opciones de internet -> Contenido -> Certificados -> Personal -> Importar...
Instrucciones para incluir un certificado personal en el Firefox: Herramientas -> Opciones -> Avanzado -> Ver certificados -> Sus certificados -> Importar...
Instrucciones para incluir un certificado personal en el Thunderbird: Herramientas -> Opciones -> Privacidad -> Seguridad -> Administrar certificados... -> Sus certificados -> Importar...
Instrucciones para incluir un certificado personal en el Outlook Express: Herramientas -> Opciones -> Seguridad -> Ids. Digitales... -> Personal -> Importar...

Para intercambiar correos cifrados con otros compañeros deberéis instalar también en vuestro gestor de correo sus certificados digitales.

Instrucciones para instalar certificados de usuario en el Thunderbird: Herramientas -> Opciones -> Privacidad -> Seguridad -> Administrar certificados... -> De otras personas -> Importar...
Instrucciones para instalar certificados de usuario en el Outlook Express: Herramientas -> Opciones -> Seguridad -> Ids. Digitales... -> Otras personas -> Importar...

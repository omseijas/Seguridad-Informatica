CA-SegI

Autoridad Certificadora para la materia Seguridad Informática

Esta autoridad certficadora ha sido desarrollada para unas prácticas de la materia Seguridad Informática de la USC. No tiene ninguna validez legal. Si no es un alumno de esta materia, por favor no emplee este servicio.

Para emplear los certificados emitidos por esta CA deberá en primer lugar instalar el certificado raíz de la misma en su navegador, gestor de correo o programa en el que vaya a utilizar dichos certificados.

Certificado raíz de esta CA en formato PEM: certificado

Instrucciones para instalarlo en Internet Explorer: Herramientas -> Opciones de internet -> Contenido -> Certificados -> Entidades emisoras raíz de confianza -> Importar...
Instrucciones para instalarlo en Firefox: Herramientas -> Opciones -> Avanzado -> Cifrado -> Ver certificados -> Autoridades -> Importar ... ó Editar -> Preferencias -> Avanzado -> Cifrado -> Ver certificados -> Autoridades -> Importar ...
Instrucciones para instalarlo en Thunderbird: Herramientas -> Opciones -> Privacidad -> Seguridad -> Administrar certificados... -> Autoridades -> Importar ...


El segundo paso será crear una solicitud de certificado según las instrucciones recibidas y enviarla a la CA para que la firme y nos devuelva el certificado.
En el plazo más breve posible la CA firmará dicho certificado y éste se publicará en esta página web (ver lista a continuación) para que tanto su propietario como cualquier otra persona lo pueda descargar.
